## General info

This project is simple Restful API that serves json replies.

## Technologies

Project is created with:

- Node.js
- Express.js
- MongoDB
- Mocha
- Chai

## Setup

To run this project, install it locally using npm:

```
$ cd /Fastech-API
$ npm install
$ npm start
```

## Test

To Test the API endpoints run the below command

```
$ npm test
```
